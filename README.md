# Leonardo Bianco — Portfolio

Personal portfolio. Editorial monochrome aesthetic with a single vermilion accent.
Static site built with **Nuxt 4** (SSG) + **@nuxt/content**, deployed to Netlify.

## Stack

- **Nuxt 4** — `nuxt generate` static output
- **@nuxt/content** — all copy is content-driven (`content/*.yml`, `content/*.md`)
- **@nuxt/fonts** — Space Grotesk, Hanken Grotesk, JetBrains Mono
- **SCSS** with design tokens (`app/assets/scss`)
- **Vitest** + `@nuxt/test-utils` for component/unit tests
- **Netlify Forms** for the contact form (no serverless functions)

## Develop

```
npm install
npm run dev        # dev server
npm test           # run the test suite
npm run generate   # static build → .output/public
```

## Content

Site copy lives in `content/`. SEO/meta, ticker and nav come from `content/settings.yml`;
each section has its own file (`hero.yml`, `now.yml`, `projects.yml`, `about.md`,
`journey.yml`, `contact.yml`). Schemas are defined in `content.config.ts`.

The Open Graph image (`public/og.png`) is generated by `node scripts/og.mjs`.

## Deploy

Netlify builds with `npm run generate` and publishes `.output/public` (see `netlify.toml`).
Contact submissions go through Netlify Forms — set the notification email in the
Netlify dashboard → Forms.
